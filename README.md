# 🔥 Habit Streak Tracker

A responsive **Habit Streak Tracker** built using **HTML, CSS, and Vanilla JavaScript**. The application allows users to create habits, mark daily progress on a 30-day calendar, track streaks, and automatically save data using the browser's LocalStorage.

---

## 🚀 Features

- Add new habits
- Mark daily progress
- 30-Day habit calendar
- Automatic streak calculation
- Delete habits
- Persistent storage using LocalStorage
- Responsive design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Browser LocalStorage API
